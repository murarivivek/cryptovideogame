OUR ROADMAP

Crypto Kotaku is a blockchain gaming website. Players can buy Video Games and sell them for profit. We have Game rooms, where players can buy Game Items and sell them. Players get dividends on Games and Game Items. In the end, players battle among themselves for profits.

  • Phase I

    Game Zone

    Players can buy games like PLAYERUNKNOWN'S BATTLE GROUNDS, DOTA, Counter Strike etc. Whenever a player buys a game, price of the game doubles. If the game is bought by someone else, the current owner gets 90% of the proceeds.

    Game owners get 10% dividends whenever someone buys Game Items from Game Rooms. Game owners will also get dividends whenever PoWs, the second tier game items are bought.

  • Phase IILaunching Now

    Game Rooms

    Game Rooms enable players to buy various Game Items related to a game. Game Items range from Heroes for DOTA to Weapon Types for Counter Strike. Whenever a player buys a Game Item, its value doubles. When the Game Item is bought by another player, Current Game Item Owner and Current Game Owner get returns. Game Item Owner gets 80% of the proceeds.

    Game Items also bring dividends. Game Item Owners get dividends whenever a PoW is bought from the Power Zone (Phase III). The dividends are distributed to the Game Item owner based on the parent of the PoW.

  • Phase III

    Power Zones

    Power Zones allow players to buy PoWs, the real heroes of Game. PoWs range from Pikachu in Pok√©mon Game Room to MAG-7 in Counter Strike Game Room. Whenever a PoW is bought, their value rises. When the PoW is bought by another player, Current PoW Owner, Current Game Item Owner and Current Game Owner get returns.

  • Phase IV

    Battle Zones

    Battle Zones are fun. This is where all the action is! Players can fight amongst each other in tournaments. Winning a Battle not only moves you up the ladder but also get you returns.
